### 1.1 Entity description
- **material**:
    - attributes:
        - id: primary key Integer, autogenerated by the database.
        - name: Unique material name value, must be sent in the request.
        - description: material description, optional in the request.
        - type: material type, must be sent in the request.
        - price: material price, must be sent in the request.
        - buy_date: when was purchased the material, this date can be sent in the request, if not, sent by default the 
      current date.
        - sold_date: this date can be sent in the request. it can be null.
        - city_code: the String foreign key to the city table, received in the request.
        - state: Indicate the status of the record, possible values: 
            - Active: the record exists
            - Inactive: the record exists but was soft deleted.
            - Available: the record is available for sale
            - Assigned: The record is reserved.
